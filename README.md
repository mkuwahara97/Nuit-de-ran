# 🌙 Nuit de ran

## デプロイ手順

### ① GitHubにアップロード
1. github.com でログイン
2. 「New repository」→ 名前をつけて作成
3. 「uploading an existing file」をクリック
4. このフォルダの**中身を全部**ドラッグ＆ドロップ
5. 「Commit changes」をクリック

### ② Vercelでデプロイ
1. vercel.com でGitHubログイン
2. リポジトリを選んで「Deploy」
3. URLが発行されます！

### ③ スマホにインストール
**iPhone** → SafariでURL開く → 共有ボタン →「ホーム画面に追加」
**Android** → ChromeでURL開く →「アプリをインストール」
